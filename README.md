# Workflow Dashboard

A simple workflow management dashboard built using HTML, CSS, and JavaScript.

## Features

- Add tasks
- Set task priority
- Add task category
- Add due date
- Mark tasks as completed
- Edit tasks
- Delete tasks
- Search tasks
- Filter by status and priority
- Sort by newest, oldest, priority, or due date
- Clear completed tasks
- Track overdue tasks
- Progress bar and dashboard statistics
- Save tasks and theme using Local Storage
- Responsive layout for mobile screens

## Tools Used

- HTML
- CSS
- JavaScript

## How to Run

1. Download the project
2. Open index.html
3. Start managing tasks
